Python

O python é uma linguagem interpretada, o que significa que ela precisa de um interpretador para que funcione. Interfaces de Desenvolvimento(IDE), funcionam como interpretadores para a linguagem python, por exemplo o Pycharm fornecido pela JetBrains, o jupyter notebook, que na verdade não é exatamente uma IDE porém isso não é relevante agora. Estrutura de Dados parte 1

A estrutura de dados de uma linguagem é a forma como podemos diferenciar o que queremos dizer para o computador. Por exemplo no português, se quisermos escrever uma frase com o intuito de ser uma pergunta, devemos usar o ponto de interrogação ao final da frase(?), do contrário a nossa mensagem não será clara, por exemplo:

  • Você vai à casa do João.
  • Você vai à casa do João?
O primeiro exemplo foi uma afirmação, já o segundo foi de fato uma pergunta, isto só pode ser identificado graças ao sinal de pontuação no final da frase.

Agora que eu acredito que tenha ficado mais claro, vamos ver a estrutura de dados do python.

String

O tipo "String" ou só "str" é um tipo bastante básico, esse tipo se refere aos dados que indicam textos, como por exemplo: casa, Marcos, água, lua, sabonete... Textos no geral. A forma de se declarar uma variável do tipo String é usando aspas simples('') ou aspas duplas("") no valor da variável

Exemplo

nome = "Anderson"

rua = "Pinheiros"

trabalho = "bancário"

Int

O tipo "int" é um tipo que se refere aos valores numericos inteiros, ou seja são os números que fazem parte do conjunto dos inteiros (Z) dentro da matemática. A forma de declarar um int é atribuindo um valor númerico à variável.

Exemplo

idade = 29

peso = 86

salario = 1550

Float

O tipo "flutuante" ou "float" atua de forma parecida com o tipo "int", ambos servem para a atribuição de valores numericos. A diferença entre o tipo int e o tipo float é que o float se trata de números do conjunto racional(Q), ou seja os números decimais. A forma de declarar uma variável do tipo float é usando valores decimais com o "." em vez da ",".

Exemplo

tamanho = 56.7

peso = 91.6

preco = 3745.90

Bool

O tipo "booleano" ou só "bool" se trata de dois valores apenas, verdadeiro e falso, este tipo é bastante usado quando desejamos realizar operações envolvendo estruturas condicionais, que também veremos mais à frente. A forma de declarar uma variável do tipo bool é usando "True" para verdadeiro, ou "False" para falso.

Exemplo

ligado = True

desligado = False

pausado = True

A princípio, estes são os tipos de dados que realmente usaremos por enquanto, ainda neste guia você verá tipos mais complexos, porém eles são apenas extensões destes já apresentados.

Neste exemplo, fizemos um programa simples para realizar a análise de voto de uma pessoa usando sua idade como parâmetro para votação segundo a legislação brasileira, ou seja, se uma pessoa tem mais de 16 anos e menos de 18 anos, 70 anos ou mais, o voto dessa pessoa é opcional. Já se alguém tem menos de 16 anos, essa pessoa não pode votar, e se alguém tem mais de 18 e menos de 70, seu voto é obrigatório.
Pensando nisso, o nosso programa faz a análise do número digitado pelo usuário e:

  • Se for menor de 16 anos, não pode votar
  • Se for maior que 18 anos e menor que 70 anos, seu voto é obrigatório
  • Caso não se encaixe em nenhum dos dois, seu voto será opcional.

Se você tem menos de 18 e mais de 16, você não se encaixa no if e nem no elif(estrutura usada quando queremos mais de duas possibilidades no programa), logo você se encaixa no else. Já se você não tem menos de 70 anos, significa que você tem mais de 70, o if e o elif são para quem tem menos de 70, e você por não ter menos de 70, é jogado no bloco else.

Loops

Os laços de repetições(loops), são estruturas que servem para repetir proçessos que podem ser cansativos ou massivos usando parâmetros parecidos com o if, else e elif. Ou seja, enquanto a condição dentro dos laços forem verdadeiras, eles realizarão a função, isso ficará mais claro logo logo. No python temos dois tipos de laços, for e while.

For

O laço do tipo for pode ser interpretado como "para cada", ou seja, para cada verdade dentro da condição, realize a tarefa.

Exemplos
Aqui nós criamos um laço for e demos uma variável para que sirva como parâmetro na execução do código, a variável i irá receber os valores no intervalo de 0 até 9, pois como explicamos antes o python desconsidera o ultimo valor (10). Em seguida pedimos para exibir o valor de i, que começou no zero e foi se atualizando no decorrer do laço, ou seja, recebendo os valores seguintes. Quando não é mais possível atualizar o valor de i devido termos atingido o limite que usamos inicialmente, o bloco é encerrado.
Aqui nós criamos um laço for e também demos uma variável para que sirva como parâmetro na execução do código, a variável i irá receber os itens da lista, do primeiro ao ultimo a partir do primeiro. Ao receber um item como seu valor, a função print irá exibir o valor recebido e em seguida i será atualizado e recebrá o próximo item da lista até que a lista acabe.
Neste caso, nós criamos um laço e também atribuimos uma variável como parâmetro, porém agora nós usamos a função len() para definir os valores para nossa variável. A função len() serve para indicar o tamanho de uma lista, ou seja quantos elementos ela possui. Trazendo isso para o nosso laço, na prática fizemos algo similar ao primeiro exemplo, porém aqui usamos o tamanho da lista(5) em vez de números pré definidos.
Aqui nós reaproveitamos a forma anterior, como a variável i recebe os indíces da lista e não os elementos, pedimos para que fosse exibido o indíce da lista que fosse correspondente ao valor de i, e como o valor da variável é o mesmo do indíce da lista ele irá imprimir o item da lista de forma similar ao segundo exemplo mostrado.
Aqui nós fizemos algo um pouco mais elaborado, uma simples taboada do número 9, acredito que você já entendeu o funcionamento do laço for e da importância de atribuir uma variável ao laço. A nossa variável recebe o valores que definimos e vai se atualizado, a medida que isso acontece ela exibe a mensagem que pedimos ou seja: 9 x {o valor de i} = {9 vezes o valor de i}
Aqui já partimos para um uso melhor do laço, criamos uma variável e seus valores serão do 1 ao 20, porém 'pulando de 2 em 2'. Como ela começa no 1, 1+2 = 3, 3+2 = 5, 5+2 = 7... até o limite de 20, ou seja i não será atualizado 20 vezes mas apenas 10 por causa da nossa condição de atualização (range(1,20,2)). Espero que isso tenha ficado claro, no decorrer do laço, pedimos que a nossa lista, inicialmente vazia, fosse recebendo o valor de i no decorrer de sua atualização. E ao terminar o laço, pedimos que fossem mostrados todos os itens da lista através da função print. Atente-se a indentação do código, pois se o print estivesse 'para dentro', a execução do código seria diferente.
Este ultimo exemplo é um reaproveitamento do anterior para mostrar que podemos ter vários laços e que podem interagir com as mesmas coisas de formas diferentes. Aqui no primeiro laço nós adicionamos o valor de i na lista, no segundo laço temos a variável j que recebe o valor do total de elementos contidos na nossa lista(10) e a medida que o valor de j se atualiza o ultimo elemento da nossa lista é removido, note que o j não aparece dentro do bloco do laço, pois ele serve apenas como chave para o laço funcionar e não interage diretamente com os elementos do laço. Ao final da execução do segundo laço, pedimos que fossem exibidos os itens da lista, porém ela está vazia já que o nosso laço removeu o ultimo item de forma consecutiva(o penultimo se torna o ultimo e por aí vai...).

While

Agora que vimos como o laço for funciona, entender o laço while não será complicado. O laço while pode ser traduzido como "enquanto", assim como o for, ele recebe uma para ser iniciado e enquanto a condição for verdadeira, ele ira repetir os comandos que forem passados. A melhor forma de aplicar o uso de cada laço é avaliando o limite da condição, por exemplo sempre usamos o for quando sabemos onde começa e onde termina uma condição. Quando sabemos onde uma condição começa, mas não sabemos onde ela termina é mais viável o uso do laço while, veremos alguns exemplos

Exemplos
Neste exemplo, nós criamos uma variável fora do laço e atribuimos a ela o valor inicial de 0, em seguida criamos um laço do tipo while e a sua condição foi de condi ser menor que 5, esse laço todo pode ser lido como "enquanto condi for menor que 5 faça:" e os comandos nesse laço são dois, o primeira é exibir o valor atual de condi, o segundo é condi receber o seu valor atual mais um(1), ou seja, o valor de condi será atualizado em uma unidade, enquanto ele não atinge o limite de 5. pois quando condi for igual a 5, a nossa condição será falsa logo o laço while será encerrado.
Aqui nós criamos a variável condi e não artibuimos valor nenhum a ela, apenas definimos que ela é do tipo str e a nossa condição do while pode ser lida como "enquanto condi for diferente da letra 'n', faça: ". E dentro do bloco do laço nós mudamos o valor de condi para um input do tipo str para que seja inserido uma letra como valor de condi, se a letra digitada for "n" a nossa condição se torna falsa e o laço é encerrado.
Aqui nós reaproveitamos o exemplo anterior e fizemos algumas mudanças na condição do while. Criamos uma lista chamada "paradas" e nela colocamos itens que irão servir como condições de parada para o laço. Condi novamente se tornou um input onde o usuário deverá digitar qualquer coisa. Caso o usuário digite algo que esteja dentro da lista "paradas", o laço será encerrado pois ele pode ser lido como "enquanto a condição não for um dos itens da lista, faça: ". O usuário digitando uma palavra contida na lista fará a condição ser falsa, logo o laço será encerrado.
Neste ultimo exemplo, nós não definimos uma condição de parada no while. Quando usamos um "True" como condição de parada, é como se estivessemos dizendo "Vai fazendo aí". Porém caso realmente não exista nada para quebrar o laço, ele será executado eternamente e dependendo do caso, o que quebra é a sua maquina. Mesmo sem ter uma condição de parada pré definida, existe sim uma condição, mas ela está dentro do próprio laço, o break. Nós pedimos que o usuário digitasse alguma coisa e usamos a estrutura if para fazer uma análise do que foi digitado. Se o que o usuário digitar for "break", o comando break, contido na estrutura if, será executado. O comando break serve para quebrar os laços independete de sua condição de parada inicial.

A primeira delas é que tuplas são imutáveis no decorrer da execução de um código. Dicionário

Entre os tipos de dados que abrigam mais de um valor, ou seja os vetores, os dicionários são os mais complexos. Um dicionário é similar a uma lista que abriga outras listas(uma matriz). Porém, nos dicionários não são usados indíces numericos para acessar seus itens, mas sim as chaves e valores. Para criarmos um dicionário temos que usar "{}" em seu valor, mas as "{}" devem conter dois elementos dentro, a chave do item e o valor do item.

Exemplo

dic = {"chave":"valor"}

As chaves devem ser o primeiro elemento e devem estar entre aspas obrigatoriamente, em seguida usamos ":" para indicar o valor da chave, que pode ser qualquer tipo de dado(str, int, listas, tuplas...)

Vamos ver alguma possibilidades que os dicionários nos proporcionam

Exibindo todas as chaves do dicionário Exibindo todos os valores do dicionário Exibindo todos os itens do dicionário Excluindo um item através de sua chave Excluindo o ultimo item do dicionário Exibindo o valor de uma chave específica Limpando o dicionário

Tratamentos de erros

Algo bastante comum dentro da programação são os erros. Os erros e bugs estão presentes no cotidiano de todos os programadores. No python, temos uma estrutura aprópriada para tratar erros e bugs, ela é composta de try(caso ocorra tudo bem) e except(caso ocorra um erro). A primeira vista, ela se parece um pouco com o if e else, mas essa estrutura serve apenas para tratamento de erros. vejá alguns exemplos.

Exemplos
Aqui nós usamos a estrutura try para definir o que queremos no nosso código. Temos uma variável texto que tem o valor do tipo string "20"; a variável numero faz a conversão do valor contido na variável texto para o tipo numerico int; em seguida nós pedimos para que fosse exibido o valor da variável numero. Caso houvesse algum erro na execução da estrutura try, seria exibido "algo deu errado", pois é o que pedimos para fazer caso houvesse erro(except)
Este exemplo condiz melhor com o que foi dito no anterior, aqui temos a variável texto que tem o valor "salada"; a variável numero tenta fazer a conversão de str par int porém ela falha pois não existem letras no tipo inteiro, apenas números. Logo o programa faz o que está contido na estrutura except.
Também é possível definir o tipo específico de erro que queremos corrigir ou tratar. Quando usamos o Execption após o except, estamos querendo tratar a exceção em si, normalmente o Execption é usado para saber o que exatamente está errado no código.
Aqui temos uma lista com 3 elementos, pedimos para que fosse exibido o elementode indíce 3 mas a lista só vai até o indíce 2, o que significa que temos um erro de indíce, então podemos trata-lo de forma individual na estrutura except. Caso o erro fosse de outro tipo, a nossa exeção não funcionaria pois ela é exclusiva para o tipo IndexError

Agora que você entende mais ou menos como funcionam os tratamentos de erro, tente praticar por conta própria e ver outros tipos de erros.
